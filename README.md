# RentalFlow

A property rental management system where property owners can list properties, manage bookings, handle maintenance requests, and generate invoices for tenants.
Core domains to be built:

- Properties (units, amenities, pricing rules)
- Bookings (reservations, availability)
- Tenants (tenant info, rental history)
- Invoices (rent, utilities, late fees)
- Maintenance (requests, scheduling, costs)


## Why build this?

I read Brent Roose's book on Laravel. Stuff used in an established company like Laravel, but then quickly forgot all about it. This project is to follow all of those patterns while building a decent sized project.
